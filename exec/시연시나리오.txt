    1. 승우 : 캘린더로 전환해서 둘러본다.
    2. 승우 : 4월 11일을 선택하고  스타벅스에 카테고리를 선택한다.
    3. 승우 : ‘스누피 + 케익’ 메모를 넣는다.
3. **승우는 리포트를 기반으로 용돈 인상 요청을 보낸다. (두 칸 모두 입력하고 바깥 터치해서 포커스 빼야 보내기 버튼 나타납니다 입력 후 빠르게 두번 눌러서 보내기 띄워주세요)**

    2. 승우 내용 : ‘5000원’, ‘버스 요금이 올랐어요’
4. 혜빈은 용돈 인상 요청에 해당하는 리포트를 읽고, 용돈 인상 요청을 수락한다.
    1. 혜빈 : 알림에서 ‘리포트 보러가기’를 누르고 각 카테고리를 살펴 본다.(1~2개)
    2. 혜빈 : 또래비교를 눌러본다.
    3. 혜빈 : 뒤로 가기를 누르고 소비 요약을 살펴 본다.
    4. 혜빈 : X버튼을 누르고 ‘수락’한다.

# 엄마 AI / 용돈 단건 요청

1. 승우는 부모님께 요청하기를 눌러 엄마 ai 설득을 시작한다.
    1. 승우 : 용돈이 필요해요 선택
    2. 승우 : ‘친구들이랑 카페 갈건데 3000원만 주세요’ 입력
    3. 승우 : ‘커피 안 먹고 요구르트 먹을게요’ 입력
    4. 승우 : 설득이 완료되면 보낸 요청 확인하기를 누른다.
2. 혜빈은 알림이 오면 용돈 요청 내역을 누른다.
    1. 혜빈 : 알림탭을 누르고 맨 위 용돈 요청 내역을 누른다.
    2. 혜빈 :  용돈 요청을 수락한다.
3. 승우는 용돈 요청이 수락되고, 송금이 완료 됐다는 알림 내역을 조회한다.
    1. 승우 : 입금 푸시알림을 확인하고, 기록장에 3000원 입금을 확인한다.
